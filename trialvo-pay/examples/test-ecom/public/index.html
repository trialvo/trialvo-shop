📘 Quick Setup Guide

  1. Open Trialvo Pay Admin and log in (or create an admin first via seed_dev.sql)
  2. Go to Services → Create a new service (slug: test_ecom, sandbox: yes)
  3. Click on the service → Generate API Key → Copy the raw key
  4. Paste the Service ID and API Key above
  5. Go to IPN/Webhooks → Create endpoint: URL = http://localhost:3456/webhooks/trialvo_pay
  6. Copy the IPN Secret and paste it above
  7. Make sure EPS sandbox credentials are set in System Config → eps
